OEM Part Number Cross-Reference Prompting

Best Practices beginner prompt-techniques Parts Lookup OEM Cross-Reference

Cross-referencing OEM part numbers across manufacturers is one of the most common tasks in the automotive aftermarket. This guide shows you how to structure your SecureAI prompts for accurate, reliable cross-reference results.

Why Prompt Structure Matters

SecureAI draws on parts databases, fitment data, and manufacturer catalogs. A well-structured prompt helps the AI narrow its search to the right part family, application, and manufacturer -- reducing false matches and saving you time.

Basic Cross-Reference Prompt

Start with the OEM part number and the target manufacturer:

What is the aftermarket equivalent of OEM part number 34-11-6-799-166
from BMW for use in a 2019 BMW X5 xDrive40i?

Key elements:

  • The full OEM part number (include hyphens or spaces as printed)
  • The OEM manufacturer
  • The vehicle year, make, model, and trim

Specifying a Target Brand

If you need a cross-reference to a specific aftermarket brand, say so directly:

Cross-reference BMW OEM brake pad set 34-11-6-799-166 to Akebono
and Bosch equivalents.

You can list multiple target brands in one prompt. SecureAI will return matches for each.

Batch Cross-Referencing

For multiple part numbers, list them clearly:

Cross-reference the following BMW OEM part numbers to their
aftermarket equivalents:

1. 34-11-6-799-166 (front brake pads)
2. 34-21-6-873-093 (rear brake pads)
3. 34-11-6-792-219 (front brake rotor)

Include Akebono, Bosch, and Wagner equivalents where available.

Numbering your parts makes it easier to match results back to your list.

Adding Context for Better Results

The more context you provide, the more accurate the cross-reference:

I need to cross-reference Toyota OEM part number 04465-48150
(front brake pad set) for a 2020 Toyota Highlander XLE AWD.
The customer wants a ceramic pad with the best NVH rating.
Include price tier if available.

Useful context to include:

  • Part category (brake pads, filters, belts, etc.)
  • Vehicle application with full YMMT (year, make, model, trim)
  • Material preference (ceramic, semi-metallic, organic)
  • Performance requirements (OE-equivalent, heavy-duty, fleet use)

Verifying Cross-Reference Results

Always verify fitment before ordering. Ask SecureAI to confirm:

Confirm that Akebono ACT1178 is a direct replacement for BMW
OEM 34-11-6-799-166 on a 2019 BMW X5 xDrive40i. Check fitment
dimensions and any known compatibility notes.

Handling Superseded Part Numbers

OEM part numbers change over time. If your number returns no results:

The OEM part number 34-11-6-761-244 may be superseded.
What is the current BMW OEM number and its aftermarket equivalents
for front brake pads on a 2015 BMW X5 xDrive35i?

SecureAI will attempt to trace the supersession chain and return the current number.

Setting Up Your Knowledge Base for Cross-Referencing

For the best cross-reference results, upload your parts catalogs and price sheets to SecureAI:

  1. Upload manufacturer catalogs (PDF) through the file upload interface. SecureAI indexes the content for lookup.
  2. Upload your price lists so SecureAI can include pricing in cross-reference results.
  3. Upload fitment guides from your aftermarket suppliers for more accurate compatibility data.

See Uploading documents for analysis for detailed upload instructions.

Prompt Templates

Copy and customize these templates for common cross-reference scenarios:

Single Part Cross-Reference

Cross-reference [OEM] part number [NUMBER] for a [YEAR] [MAKE]
[MODEL] [TRIM]. Show aftermarket equivalents from [BRAND1],
[BRAND2], and [BRAND3].

Bulk Cross-Reference

Cross-reference these [OEM] part numbers for a [YEAR] [MAKE]
[MODEL] [TRIM]:

1. [NUMBER1] ([DESCRIPTION1])
2. [NUMBER2] ([DESCRIPTION2])
3. [NUMBER3] ([DESCRIPTION3])

Show all available aftermarket equivalents with brand names.

Supersession Lookup

Part number [NUMBER] from [OEM] may be superseded. Find the
current part number and aftermarket equivalents for [PART TYPE]
on a [YEAR] [MAKE] [MODEL].

Tips for Reliable Results

  • Always include the vehicle application. The same OEM number can have different aftermarket equivalents depending on the vehicle.
  • Use the full part number with hyphens or formatting as printed on the part or in the catalog.
  • Specify the part category when the number alone might be ambiguous.
  • Ask for confirmation before ordering -- request fitment dimensions or compatibility notes.
  • Upload your catalogs to give SecureAI the best data to work with.
